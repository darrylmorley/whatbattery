# WhatBattery 1.1

A free update with two additions: WhatBattery now keeps itself up to date, and you can set the text size to suit you. Both are free. macOS 14 and later, Apple Silicon.

## New

- **Automatic updates.** WhatBattery now checks for new versions on its own and offers them in a banner, with a one-click install that verifies and replaces the app for you. You can also check any time from the menu or Settings, and turn the update notifications off if you prefer. Every update is signed with a Developer ID and notarised by Apple, and WhatBattery only installs a build signed by the same developer as the one you are running.
- **Adjustable text size.** A new slider in Settings scales the text across every panel, from the menu bar popover to the main window, so you can make it larger to read at a glance or smaller to fit more on screen.

Read-only as always: WhatBattery never changes how your devices charge.
